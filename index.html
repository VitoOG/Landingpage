Engineering Manifesto

01

Architecture Before Implementation

System-first thinking. Explicit state ownership and explicit control loops. Production constraints ALWAYS precede theoretical elegance.

02

Deterministic Behavior Over Heuristics

Zero tolerance for hidden state. ONLY explicit data flow. Multiple executions MUST produce identical state without accumulation.

03

Failure-First Design

Assume runtime failure. Isolate faults immediately. Catch and lock return codes. ALWAYS provide a predictable degraded state (Safe Shutdown).

04

Single Source of Truth (SSOT)

One meaning = one file. Change by overwrite ONLY. NEVER generate versions (v2, final). Git stores history; the filesystem stores active state.
